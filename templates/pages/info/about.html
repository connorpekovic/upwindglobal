This project is here to serve as the perfect example as to how to make a highly secure database driven web app. The source code is on GitHub. This domain is for sale.

This website uses state of the art web technology called the Python Django Web Application Framework that runs in a low-cost, highly secure Amazon Web Services Elastic Beanstalk deployment.

This website implimentes all the fundamental functions of 12 Factor web development; containerization, consuming JSON data with REST API's, data visualization, and forms to interact with data. The codebase is small, readable, and well commented to serve as a perfect example for future web projects for myself and anyone on the open internet. The code base will be forever available here on GitHub.

Future plans for this website include enriching the data provided in return for a location. This project will be used as an example for future tutorials.
